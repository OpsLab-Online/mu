### 介绍

热榜聚合这个产品已经有很多很多了。我只是按照自己的想法写了个自己用的而已。

主要是为了学习Go开发。 

目前使用到的技术栈

+ Go
+ Goland
+ Vue.js
+ Drone CI
+ Docker

### 安装

```shell
make crawler # 编译cron，抓取数据
make mu # 编译前端展示
```
